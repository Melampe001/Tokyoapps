Carpeta de imágenes para la ruleta.

Coloca aquí tus imágenes de ruleta personalizadas o descargadas (por ejemplo, desde Unsplash).

Ejemplos de nombres:
- ruleta_001_a.jpg
- ruleta_001_b.png
- ruleta_001_c.webp

Asegúrate de referenciar estas imágenes en tu código Flutter y en pubspec.yaml:
```
flutter:
  assets:
    - assets/ruleta 001/
```
